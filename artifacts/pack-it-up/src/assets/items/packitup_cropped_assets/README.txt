Pack It Up cropped pixel assets

Generated from the five source sheets provided in chat.

Folders:
- normalized/: game-ready transparent PNGs centered on 64/96/128/etc canvases
- raw_crops/: tight transparent crops
- by_room/: copies grouped by room/category
- contact_sheets/: visual review sheets

Files:
- manifest.csv / manifest.json: source sheet, row, item, label, usage, canvas size
- error_report.txt: crop errors, if any

Notes:
- Cropped only; no image generation was used.
- Assets were normalized individually because the source sheets use uneven spacing.
- Tiny stray label fragments were removed by connected-component cleanup and one manual crop override.

Self-identification · before touching anything

  1. Harness — Cursor / Codex / ChatGPT / Claude / OpenRouter / unknown?
  2. Model — Grok / Composer / Sol / Luna / Fable / Opus / GLM / unknown?
  3. Permissions — inspect repo? edit repo? run tests? paid API?
  4. Task risk — tiny / routine / structural / taste / debugging / unknown?
If any answer is unknown, ask:

  "Before I act, which harness/model
   am I right now?"

If the task is outside your role,
return a handoff instead of forcing it:

  Recommended handoff: [model/harness]
  Reason:              [why]
  Risk level:          [tiny…taste]
  Code edits allowed:  [yes/no/inspect]
  Prompt:              [paste-ready]

Budget · one line to remember

1 · Cursor native first 2 · ChatGPT Sol for thinking 3 · Codex when the repo needs a real agent 4 · Claude for taste & review 5 · OpenRouter only as approved overflow

"Do not spend expensive intelligence moving furniture. Do not let cheap labor redesign the house."

If you do not know who you are, ask. If the task is outside your role, hand it off. If the task can break the house, do not pretend it is moving the radio.
